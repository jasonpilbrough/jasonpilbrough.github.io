40kHz sonar imaging in air

Design project based on using ultrasound to accurately image the surrounding environment. Lead a team of four during the initial phase of the project to develop a sonar capable of 1D imaging and achieved first place in a design challenge. Extended the functionality of the sonar to provide 2D imaging capability individually as a personal project.

Scope of the project includes the design of necessary analogue electronics, embedded systems development in C++, digital signal processing algorithm implementation in Python, frontend development in HTML, CSS and JavaScript using the Bootstrap 4 library, and backend development in Python using the Flask framework.

Handwritten Digit Recognition

Project that uses machine learning to recognise and classify handwritten digits. A neural network was built in Python from first principles and trained using the MNIST dataset. The model was able to correctly classify digits with an accuracy of over 95%. The neural net was ported to run in JavaScript in the browser. The required image pre-processing was also implemented in JavaScript.

Bookshop App

Desktop application designed to manage the activities of a bookshop and replace existing low-tech solutions with a more efficient digital workflow. Developed in Java with a MySQL backend. Provides integrated management of bookshop and library entities and tracks all book/audio/video resources, loans, members, and sales.

Notable features include generation of cash-up and sales reports; user management with hierarchical access control; error logging and reporting mechanism; remote diagnostics and deployment capabilities.
