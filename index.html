40kHz sonar imaging in air

Design project based on using ultrasound to accurately image the surrounding environment. Lead a team of four during the initial phase of the project to develop a sonar capable of 1D imaging and achieved first place in a design challenge. Extended the functionality of the sonar to provide 2D imaging individually as a personal project.

Scope of the project including the design of necessary analogue electronics, embedded systems development in C++, creation of a digital signal processing algorithm suite in Python, frontend development in HTML, CSS and JavaScript using the Bootstrap 4 library, and backend development in Python using the Flask framework.

Demonstration of how a machine learning can be used to recognise handwritten digits. A neural network was built in Python from first principles and trained with the MNIST dataset using stochastic gradient descent. The neural net was ported to run in JavaScript in the browser.

Bookshop App

Desktop application designed manage the activities of a bookshop and replace existing low tech solutions with a more efficient digital workflow. Developed in Java with a MySQL backend. Provides integrated management of bookshop and library entities and tracks all book/audio/video resources, loans, members, and sales.

Notable features include generation of cash-up and sales reports; user management with hierarchical access control; error logging and reporting mechanism; remote diagnostics and deployment capabilities.

Envirologger

Internet of things (IoT) solution to monitoring the conditions in a greenhouse using a Raspberry Pi and Python. Project involved digital circuit design and construction to meet hardware requirements and required use of communication protocols including I2C and SPI to interface with external sensors. Integration with the Blynk platform was done to provide remote access to the system. Project was delivered with a technical report.
