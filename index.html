40kHz sonar imaging in air

Research project tackling a challenging engineering design problem with large scope. Designed analogue electronics including filters, amplifiers, and regulators. Wrote low-level microcontroller code in C with careful consideration of stringent sampling requirements. Developed a suite of digital signal processing algorithms in Python to processes received signals, and construct the sonar image. Developed a web interface using HTML, CSS, Javascript, and the Bootstrap 4 library. Served the web application with a Python backend using the Flask framework.

Lead a team of four members during the initial phase of the project to develop a sonar capable of 1D imaging. Received first place in design competition having achieved the best sonar imaging performance. Extended the functionality of the sonar to provide 2D imaging individually as a personal project.

Envirologger

Internet of things (IoT) solution to monitoring the conditions in a greenhouse using a Raspberry Pi and Python. Required use of communication protocols including I2C and SPI to interface with external sensors. Integration with the Blynk platform to provide remote access. Digital circuit design and construction was required to meet hardware requirements.

Non-technical skills required for this project included effective communication and division of tasks with project partner, presentation and speaking skills, and report writing for submission of technical document.
